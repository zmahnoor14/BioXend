## 6. Minimum Required Information Checklist

### 6.1 Xenobiotic Compound (Level A - Essential)

- [ ] Chemical structure (SMILES, InChI, SDF)
- [ ] Compound identifier (ChEMBL, PubChem, or ChEBI ID)
- [ ] InChI Key
- [ ] Molecular formula
- [ ] Molecular weight
- [ ] Chemical name (IUPAC or common)
- [ ] ChEBI classification
- [ ] ClassyFire chemical taxonomy

### 6.2 Microorganism (Level A - Essential)

- [ ] Scientific name (Genus species)
- [ ] NCBI Taxonomy ID
- [ ] Strain designation
- [ ] Growth medium composition
- [ ] Growth temperature
- [ ] Oxygen requirement (aerobic/anaerobic)
- [ ] Growth phase at harvest

### 6.3 Experimental Conditions (Level A - Essential)

- [ ] Substrate concentration with units
- [ ] Incubation time with units
- [ ] Temperature
- [ ] pH (if measured)
- [ ] Cell density or biomass
- [ ] Sample type (culture, lysate, supernatant)

### 6.4 Biotransformation Result (Level A - Essential)

- [ ] Transformation observed (yes/no)
- [ ] Product identification (if applicable)
- [ ] Analytical method used
- [ ] Quantification method
- [ ] Biological replicates (n≥3 recommended)

### 6.5 Data Provenance (Level A - Essential)

- [ ] Publication reference or dataset DOI
- [ ] Author list and affiliations
- [ ] Date of experiment
- [ ] Data license (e.g., CC0, CC-BY-4.0)
- [ ] Contact information

### 6.6 Enhanced Metadata (Level B - Recommended)

- [ ] Gene/enzyme responsible (if known)
- [ ] Transformation mechanism
- [ ] Kinetic parameters (Km, Vmax, if measured)
- [ ] Product structure confirmation (Level 1 MSI)
- [ ] Raw data deposition (mzML, nmrML)
- [ ] Computational analysis (molecular docking, etc.)

### 6.7 Optional Information (Level C - Useful)

- [ ] Time-course data
- [ ] Dose-response curve
- [ ] Comparative analysis across strains
- [ ] Environmental factors tested
- [ ] Genetic modification details
- [ ] Protein expression data
